# Security Policy

## Supported Versions

Strategos is an early MVP under active development and does not yet have tagged releases. Security fixes are provided only for the latest code on the `main` branch.

| Version        | Supported          |
| -------------- | ------------------ |
| `main` (latest)| :white_check_mark: |
| older commits  | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.

Instead, please use GitHub's private vulnerability reporting for this repository:

1. Go to the repository's Security tab.
2. 2. Select "Report a vulnerability".
   3. 3. Provide a clear description of the issue, steps to reproduce, affected version/commit, and the potential impact.
     
      4. You can also open a report directly at https://github.com/BigBugaboo/strategos/security/advisories/new
     
      5. We aim to acknowledge new reports within a few days and will keep you updated as the issue is triaged and resolved. Once a fix is available, we will coordinate an appropriate disclosure timeline with the reporter.
